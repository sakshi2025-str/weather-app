# weather-app
A responsive Weather Application built with React that provides real-time weather updates using API integration . Implemented city search functionality, dynamic UI redering with React hooks, and clean component-based architecture.

## Features
- Search weather by city name
- Real-time temperature and humidity
- clean and responsive UI

## Tech Stack
- React
- JavaScript
- HTML
- CSS
- Open Weather API

## How to Run
1- Clone the responsitory
2- Run "npm install"
3- Run "npm run dev"
